## SpotifyTrack

Represents a Spotify track with complete metadata.

<ResponseField name="id" type="string" required>
  Unique Spotify identifier for the track
</ResponseField>

<ResponseField name="name" type="string" required>
  The name of the track
</ResponseField>

<ResponseField name="type" type="'track'" required>
  The object type, always set to 'track'
</ResponseField>

<ResponseField name="artists" type="SpotifyArtist[]" required>
  Array of artists who performed the track
</ResponseField>

<ResponseField name="album" type="SpotifyAlbum" required>
  The album on which the track appears
</ResponseField>

<ResponseField name="available_markets" type="string[]" required>
  List of country codes where the track is available
</ResponseField>

<ResponseField name="disc_number" type="number" required>
  The disc number of the track (usually 1)
</ResponseField>

<ResponseField name="duration_ms" type="number" required>
  The track length in milliseconds
</ResponseField>

<ResponseField name="episode" type="boolean" required>
  Whether the track is an episode (podcast)
</ResponseField>

<ResponseField name="explicit" type="boolean" required>
  Whether the track has explicit content
</ResponseField>

<ResponseField name="href" type="string" required>
  Full Spotify API URL for the track
</ResponseField>

<ResponseField name="is_local" type="boolean" required>
  Whether the track is from a local file
</ResponseField>

<ResponseField name="preview_url" type="string | null" required>
  URL to a 30-second preview of the track, or null if not available
</ResponseField>

<ResponseField name="track" type="boolean" required>
  Whether the item is a track
</ResponseField>

<ResponseField name="track_number" type="number" required>
  The track number on the album
</ResponseField>

<ResponseField name="uri" type="string" required>
  Spotify URI for the track (spotify:track:...)
</ResponseField>

<ResponseField name="is_playable" type="boolean">
  Whether the track can be played in the current market
</ResponseField>

## SpotifyImage

Represents an image resource from Spotify.

</ResponseField>

</ResponseField>

</ResponseField>
